BUTTONS:
- Primary: Professional blue (#0066cc), 40px height, subtle shadow, hover lift
- Secondary: Gray outline (#6c757d), transparent background
- Danger: Alert red (#d63031), reserved for critical actions
- Ghost: Transparent with minimal border, text-only emphasis

CARDS:
- Glass panel: rgba(255,255,255,0.08), backdrop-blur, 1px border
- Rounded corners: 12px for cards, 8px for smaller elements
- Elevation: 0-3 levels using box-shadow
- Padding: 24px standard, 16px compact

STATUS INDICATORS:
- Success: Green (#00b894) with checkmark
- Warning: Amber (#fdcb6e) with alert icon
- Error: Red (#d63031) with X mark
- Pending: Gray (#95a5a6) with clock
- Active: Blue (#0066cc) with pulse animation

TYPOGRAPHY:
- H1: 32px/40px SF Pro Display, semibold
- H2: 24px/32px SF Pro Display, semibold
- Body: 16px/24px Inter, regular
- Caption: 14px/20px Inter, regular
- Code: 14px/20px JetBrains Mono, monospace

SPACING SYSTEM:
- Base unit: 8px
- Scale: 4, 8, 16, 24, 32, 48, 64, 96px

DESIGN TOOLS:
- Use with: Figma, Adobe XD, Sketch, or AI tools (Midjourney, DALL-E, Google Stitch)
- Specify: "Professional exam interface", "Enterprise SaaS design", "Dark theme"
- Quality: Request 4K resolution for detailed mockups

ANIMATION GUIDELINES:
- Duration: 200-400ms for micro-interactions, 600ms for major transitions
- Easing: Use ease-out for entrances, ease-in for exits
- Trigger: On hover, focus, click - maintain 60fps performance

ACCESSIBILITY REQUIREMENTS:
- WCAG 2.1 Level AA compliance minimum
- Color contrast ratio: 4.5:1 for normal text, 3:1 for large text
- Keyboard navigation: Full support with visible focus indicators
- Screen readers: Proper ARIA labels and semantic HTML
- Font scaling: Support up to 200% zoom
- Motion: Respect prefers-reduced-motion settings

RESPONSIVE BREAKPOINTS:
- Desktop: 1920px (optimal), 1440px, 1280px
- Tablet: 1024px, 768px
- Mobile: 414px, 375px (exam mode desktop-only, dashboard responsive)

BROWSER SUPPORT:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- Electron 28+ (primary platform)
